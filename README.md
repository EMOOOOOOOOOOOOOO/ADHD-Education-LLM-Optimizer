# ADHD-Education-LLM-Optimizer
專為 ADHD 與 ASD 學生設計的閱讀與認知負荷優化系統

1. 專案背景
針對 ADHD (注意力缺陷過動症) 或 ASD (自閉症譜系障礙) 學生在面對長難句或艱澀詞彙時的認知負荷問題，本專案開發了一套基於大語言模型 (LLM) 的題目敘述優化系統。

2. 核心技術亮點 

提示詞工程 (Prompt Engineering)：精心設計 4 大任務邏輯，包含詞彙簡化、長句拆解、公平性保持（同類異構）及語意順暢度校對。

LLM 整合應用：串接 OpenAI gpt-4o 模型，實現高效能的語意理解與自動化改寫。

後端架構：使用 Python Flask 框架建置 API 服務，支援 JSON 格式請求與回應。

自動化結構解析：實作自定義解析算法 (parse_gpt_response)，將 AI 生成的非結構化文字自動轉化為結構化數據。

3. 系統功能
詞彙替代建議：自動辨識難詞並提供國小至高中程度的替代詞。

句子簡化：將複雜長句改寫為易讀短句，同時維持核心考點不變，確保教育公平性。
